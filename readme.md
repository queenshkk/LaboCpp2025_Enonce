# Garage Management System (C++)
Application de gestion d'un garage automobile réalisée en C++.

## Fonctionnalités
- Gestion des voitures
- Gestion des clients
- Gestion des employés
- Calcul du prix avec options

## Technologies utilisées
- C++

## Compilation
g++ *.cpp

## Lancer le programme
./ApplicGarage
